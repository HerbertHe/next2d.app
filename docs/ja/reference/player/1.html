DisplayListとDisplayObjectとDisplayObjectContainerの関係性

DisplayList

DisplayListは大きく3つの要素から構成されています。Stage、DisplayObjectContainer、DisplayObjectの3つの主要な要素で構成されています。

Stage

Stageは全てのDisplayObjectを保持する親オブジェクトです。

DisplayObjectContainer

DisplayObjectContainerはDisplayObjectを配置することができるDisplayObjectです。DisplayObjectContainerを継承するクラスは以下の4つです。

  • Loader
  • MovieClip
  • Sprite
  • Stage

DisplayObject

DisplayObjectは描画処理を実行するオブジェクトです。DisplayObjectを継承するクラスは以下の9つです。

  • DisplayObjectContainer
  • Loader
  • MovieClip
  • Sprite
  • Stage
  • InteractiveObject
  • Shape
  • TextField
  • Video

Adding Objects

DisplayObjectContainerにDisplayObjectを追加するには、DisplayObjectContainerのaddChildメソッドを使います。
