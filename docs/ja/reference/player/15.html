scale9Grid

scale9Grid is a property that sets the grid for scaling up/down 9 slices. The default value is null, and the Rectangle object is assigned when setting it.

Top, bottom, left, right and center area => zoom in/out as usual
Top, bottom, left and right corners => fixed without enlargement/reduction
Top, bottom, left and right center area => magnify/reduce only in left/right direction
Top and bottom center area of left and right => zoom in/out only in up/down direction

Example

const {Rectangle} = next2d.geom;
const {Shape, Sprite} = next2d.display;

const shape = new Shape();
shape
    .graphics
    .beginFill()
    .drawRoundRect(0, 0, 60, 60, 10);

const sprite = new Sprite();
sprite.addChild(shape);
sprite.scale9Grid = new Rectangle(10, 10, 40, 40);
