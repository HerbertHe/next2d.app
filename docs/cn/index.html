Player

NoCode Tool

这是一个在网络浏览器上运行的创作工具，主要用于NoCode开发，你可以直观地创建你所想象的动画，导出的数据可以通过上传到网络轻松发布。

Framework

Roadmap

玩家特点

旨在提供一个快速的2D播放器。

开放源码

Next2D Player作为开放源码软件免费提供给个人和商业使用。

快速渲染

体验轻量级、快速、漂亮的抗锯齿渲染。

完全支持WebGL

移植了 "swf2js "的仿真技术，以实现类似Flash的渲染，如WebGL中的Filter和Shape。

各种文本功能

包括各种文本功能，如固定文本、动态文本输入和HTML文本。

多平台支持

只需一段代码，就可以从桌面到移动端提供互动的、具有视觉吸引力的内容。

浏览器支持

它与最新的桌面和移动浏览器具有互操作性，并经过全面测试，在尽可能广泛的浏览器中提供性能和可靠性。

API

广泛的文档、示例代码和初步介绍的教程使新老开发者都能轻松地迅速上手。

MP3/MP4

支持阅读音频和视频，包括MP3和MP4。

NoCode工具的特点

一个在网络浏览器上运行的创作工具。

没有代码

你可以直观地创建你心目中的动画，而无需编写任何代码，导出的数据可以很容易地通过上传到网络上发布。

不需要安装

它是一种网络服务，任何访问它的人都可以立即使用。

转型

使用鼠标，你可以很容易地将内容向上、向下、向左、向右、变换、放大和缩小、旋转等，并增加和减少颜色。

混合模式

视觉效果，如加法、减法、乘法和反转，可以用鼠标添加或删除，而且这些都可以很容易地进行操作。

过滤器

诸如模糊、发光、阴影、渐变等滤镜可以用鼠标轻松添加或删除，而且可以轻松操作。

缓和

缓和功能允许你使你的动画更加逼真和动态。

输出

可以导出为JSON、WEBM、APNG和GIF动画，JSON可以由Next2D Player读取和绘制。

SWF

虽然它很简单，但也可以读取SWF文件。

框架特点

它是一个框架，通过固定开发模式简化了可读性和可共享性。

开放源码

Next2D框架作为开放源码软件免费提供给个人和商业使用。

快速启动

你可以通过简单地执行以下命令来创建一个应用程序。
[Command]

npx create-next2d-app app-name

MVVM模式

MVVM（Model-View-ViewModel）模式，着眼于UI设计师和程序员合作创建应用程序的未来。

团队发展

固定的开发模式（MVVM），可读性和简化的共享。它允许舒适的团队开发。

与NoCode工具的合作

与Flash时代一样，符号管理是可能的，允许动画师和程序员创建没有冲突的应用程序。

单页应用

现在可以通过URL来管理场景，这在以前的Canvas/WebGL应用程序中是很困难的。

输出

多个环境变量可以很容易地被导出并合并到一个文件中。

文档/教程

广泛的文档和教程支持你从学习到实际使用。
