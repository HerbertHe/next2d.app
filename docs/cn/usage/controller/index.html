Controller Area

$This area can be used to manipulate DisplayObjects in the Screen Area, such as scaling, rotation, color manipulation, BlendMode, and Filter. The Library tab allows you to load external data (images, MP4, MP3, SWF), and the JavaScript tags allow you to view and edit a list of JavaScript tags embedded in the entire project. You can also check and edit the list of JavaScript tags embedded in the entire project.$

Stage Setting

Set the width, height, frame rate, and background color of the display area (Stage).

Object Setting

Set the name and symbol of the display object. The name can be accessed with this.[[name]] in javaScript, and the symbol can be used as a class.

Transform

Controls the width, height, scaling, and rotation of the displayed object.

Color Transform

Add and subtract red, green, blue, and transparency.

read more

Blend Mode

Blending mode can be used to create compound images. "Compounding" is the process of changing the transparency or color interaction of multiple overlapping objects. Blending allows you to blend colors in overlapping movie clips to create unique effects.

read more

Filters

Filters can be used to add high quality visual effects to text, and movie clips. You can also use motion tweening to animate the filter you apply.

read more

Library

The library can load images (jpeg, gif, png), sounds (mp3), videos (mp4), and SWF files as external data. Vector drawings, text, and movie clips created on the screen can also be managed.
