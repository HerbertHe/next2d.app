Screen Area

在这个区域，你可以检查绘图信息，如当前帧中显示在舞台上的DisplayObjects的查看、放置和堆叠顺序。你可以通过按Command+Enter或Ctrl+Enter来预览和播放播放器。

标签

你可以添加或删除新项目，或指定你目前正在进行的项目。

对齐

你可以将选定的显示对象水平或垂直地对齐。 在垂直方向上，所选的显示对象可以在所选显示对象的右边、中间或左边对齐，在水平方向上，它们可以在所选显示对象的顶部或底部对齐。
