DisplayList，DisplayObject，和DisplayObjectContainer

  1. Home
  2. Reference
  3. Chapter 01

DisplayList

DisplayList由三个主要元素组成。舞台"、"DisplayObjectContainer "和 "DisplayObject"。

Stage

'舞台'是一个容纳所有'DisplayObject'的对象。

DisplayObjectContainer

'DisplayObjectContainer'是一个'DisplayObject'，可以用来放置DisplayObjects，它继承于以下四个类。

  • Loader
  • MovieClip
  • Sprite
  • Stage

DisplayObject

'DisplayObject'是一个执行绘图过程的对象。从'DisplayObject'继承的9个类如下。

  • DisplayObjectContainer
  • Loader
  • MovieClip
  • Sprite
  • Stage
  • InteractiveObject
  • Shape
  • TextField
  • Video

Adding Objects

要将一个 "DisplayObject "添加到一个 "DisplayObjectContainer "中，请使用DisplayObjectContainer的addChild方法。
