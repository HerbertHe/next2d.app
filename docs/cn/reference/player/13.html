Reference and change DisplayObject properties "mask".

mask

Set the DisplayObject to be the mask of the DisplayObject. The default value is null. To set the mask, assign a DisplayObject. To remove the mask, assign null.

Example

const {Shape, BitmapData, Sprite} = next2d.display;
const image = new Image();
image.src = "[[image path]]";

const bitmapData = new BitmapData();
bitmapData.image = image;

const shape = new Shape();
shape
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

const sprite = new Sprite();
sprite.addChild(shape);

const maskShape = new Shape();
maskShape
    .graphics
    .drawCircle(0, 0, 50);

sprite.mask = maskShape;
