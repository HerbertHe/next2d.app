GlowFilter

GlowFilter is a class that stores information about glow effects. Objects can be generated by specifying the value of each property in the constructor arguments, or by default with no arguments, and only properties that need to be changed are reassigned.

  • color: The color of the glow.
  • alpha: The alpha transparency value for the color.
  • blurX: The amount of horizontal blur.
  • blurY: The amount of vertical blur.
  • strength: The strength of the imprint or spread.
  • quality: The number of times to perform the blur.
  • inner: Specifies whether the glow is an inner glow.
  • knockout: Specifies whether the object has a knockout effect.

Example

const {GlowFilter} = next2d.filters;
const {Shape} = next2d.display;

const shape = new Shape();
shape
    .graphics
    .beginFill()
    .drawRect(0, 0, 50, 50);

shape.filters = [new GlowFilter()];
this.addChild(shape);
