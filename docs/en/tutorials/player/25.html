When the mouse pointer enters/exits the area.

  1. Home

MOUSE_OVER/MOUSE_OUT/ROLL_OVER/ROLL_OUT

The MOUSE_OVER event is sent when the mouse pointer enters the hit area of a MovieClip, TextField, Sprite, Loader, or Stage object, and the MOUSE_OUT event is sent when the mouse pointer goes out of the hit area. The ROLL_OVER and ROLL_OUT events are also MouseEvents that are sent out when the mouse pointer enters or leaves the hit area. The difference between them lies in the difference in the child display objects in the events fired in the DisplayObjectContainer where the event listener is registered: the MOUSE_OVER and MOUSE_OUT events distinguish multiple child display objects in the container, and the ROLL_OVER and ROLL_OUT events are sent when the child display object The MOUSE_OVER and MOUSE_OUT events distinguish between multiple child display objects in a container, and an event is fired each time the mouse is moved between them. On the other hand, the ROLL_OVER and ROLL_OUT events do not distinguish between multiple child display objects and consider them identical. Therefore, mouse movement between child display objects will not generate an event.

Example

const {MouseEvent} = next2d.events;

// When the mouse pointer enters the hit area
this.addEventListener(MouseEvent.MOUSE_OVER, (event) => {}));
this.addEventListener(MouseEvent.ROLL_OVER, (event) => {}));

// When the mouse pointer leaves the hit area
this.addEventListener(MouseEvent.MOUSE_OUT, (event) => {}));
this.addEventListener(MouseEvent.ROLL_OUT, (event) => {}));
