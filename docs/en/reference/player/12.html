Reference and change DisplayObject properties "alpha".

  1. Home

alpha

A property that represents the transparency of a DisplayObject. The data type is Number and valid values are 0.0 to 1.0.

Example

const {Shape, BitmapData} = next2d.display;
const image = new Image();
image.src = "[[image path]]";

const bitmapData = new BitmapData();
bitmapData.image = image;

// alpha 1
const shape1 = new Shape();
shape1
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);
shape1.alpha = 1.0;

// alpha 0.5
const shape2 = new Shape();
shape2
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);
shape2.alpha = 0.5;
