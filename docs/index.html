Player Features.

Aims to provide a fast 2D player.

Open Source

Next2D Player is available free of charge as open source software for personal and commercial use.

Fast

Experience lightweight, fast, and beautiful anti-aliased rendering.

Full WebGL support

emulation technology of "swf2js" is ported to realize Flash-like rendering of Filter, Shape, etc. in WebGL.

Various Text Functions

Various text features such as fixed text, dynamic text input, and HTML text are included.

Multi-Platform Support

Deliver interactive and visually appealing content from desktop to mobile with a single code base.

Browser Support

It is interoperable with the latest desktop and mobile browsers and has been thoroughly tested to deliver performance and reliability across the widest possible range of browsers.

API

Extensive documentation, sample code, and tutorials for initial introduction make it easy for new and old developers to get up to speed quickly.

MP3/MP4

Supports reading audio and video, including MP3 and MP4.

NoCode Tool Features.

2D entertainment development made easy! Fast! Comfortable.

No Code

You can intuitively create the animations you envision without writing code.

No installation required

It is a web service that can be used immediately by anyone who accesses it.

Transform

Using the mouse, you can easily move the content up, down, left, right, transform, zoom in and out, rotate, etc., and add and subtract colors.

Blend Mode

Visual effects such as addition, subtraction, multiplication, and inversion can be easily manipulated with the mouse.

Filter

Filters such as blur, glow, drop shadow, and gradient can be easily added with the mouse, and layers are supported.

Easing

The easing function allows you to make your animations more realistic and dynamic.

Output

Can be exported to JSON, WEBM and Animation GIF.

SWF

Although it is simple, it is also possible to read SWF files.

Framework Features.

Support systematic team development.

Open Source

Next2D Framework is available free of charge as open source software for personal and commercial use.

Quick Start

You can create a Next2D app by simply executing one command without any build settings.
[Command]
npx create-next2d-app app-name

MVVM pattern

Model-View-ViewModel (MVVM) pattern, looking toward a future where UI designers and programmers collaborate to create applications.

team development

Fixed development pattern (MVVM) for readability and simplified sharing.

Work with NoCode Tool

As in the Flash era, symbol management is possible, allowing animators and programmers to create applications without conflicts.

Single Page Application

It is now possible to manage scenes by URL, which has been difficult with previous Canvas/WebGL applications.

OUTPUT

You can easily export each environment and consolidate the files into one.

Documentation/Tutorials

Extensive documentation and tutorials to support you from initial learning to practical use.
