scaleX/scaleY

Indicates the scaling ratio of the DisplayObject's width (scaleX) and height (scaleY). A value of 0.0 to 1.0 represents a scale of 0 to 100%; to scale a DisplayObject, assign a value to this property. To scale a DisplayObject, assign a value to this property. A negative value will flip the display object left to right or up and down.

Example

const {Shape, BitmapData} = next2d.display;
const image = new Image();
image.src = "[[image path]]";

const bitmapData = new BitmapData();
bitmapData.image = image;

// scaleX 1.5
const shape1 = new Shape();
shape1
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape1.scaleX = 1.5;

// scaleY 1.5
const shape2 = new Shape();
shape2
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape2.scaleY = 1.5;

// scaleX ans scaleY 1.5
const shape3 = new Shape();
shape3
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape3.scaleX = 1.5;
shape3.scaleY = 1.5;
