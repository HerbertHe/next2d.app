Reference and change DisplayObject properties 'rotation'.

  1. Home
  2. Tutorials
  3. Day 10

rotation

rotation angle of the DisplayObject. The unit is degrees (°). Negative values and values greater than 360° can be specified, but internally they will be replaced with the range of 0 to 360°.

Example

const {Shape, BitmapData} = next2d.display;
const image = new Image();
image.src = "[[image path]]";

const bitmapData = new BitmapData();
bitmapData.image = image;

// rotation 45
const shape1 = new Shape();
shape1
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape1.rotation = 45;

// rotation -45
const shape2 = new Shape();
shape2
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape2.rotation = -45;

// rotation 405
const shape3 = new Shape();
shape3
    .graphics
    .beginBitmapFill(bitmapData)
    .drawRect(0, 0, bitmapData.width, bitmapData.height);

shape2.rotation = 405;
