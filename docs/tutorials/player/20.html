Convert between global and local coordinates.

  1. Home
  2. Tutorials
  3. Day 20

globalToLocal/localToGlobal

globalToLocal method converts the global coordinates of the argument Point object to local coordinates, and conversely, the localToGlobal method converts the local coordinates to global coordinates.

Example

const {Shape, Sprite} = next2d.display;
const {Point} = next2d.geom;

const sprite1 = new Sprite();
const sprite2 = new Sprite();
const sprite3 = new Sprite();
const shape1 = new Shape();
const shape2 = new Shape();
const shape3 = new Shape();

shape1
    .graphics
    .lineStyle()
    .drawRect(0, 0, 200, 150);

sprite1.addChild(shape1);

shape2
    .graphics
    .lineStyle()
    .drawRect(0, 0, 160, 110);

sprite2.x = 30;
sprite2.y = 30;
sprite2.addChild(shape2);

shape3
    .graphics
    .lineStyle()
    .drawRect(0, 0, 90, 60);

sprite3.x = 60;
sprite3.y = 30;
sprite3.addChild(shape3);

sprite2.addChild(sprite3);
sprite1.addChild(sprite2);
this.addChild(sprite1);

console.log(sprite3.globalToLocal(new Point(90, 60))); // 0, 0
console.log(sprite3.localToGlobal(new Point(0, 0))); // 90, 60
