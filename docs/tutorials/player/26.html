MOUSE_DOWN/MOUSE_UP

The MOUSE_DOWN event is a MouseEvent sent out when the mouse button is pressed on a MovieClip, TextField, Sprite, Loader, or Stage object, or when a tap is started, and the MOUSE_UP event is sent out when the mouse button that was pressed on an object is released, or when a tap is ended.

Example

const {MouseEvent} = next2d.events;

// When a mouse button is pressed in the area, or when a touch start is made.
this.addEventListener(MouseEvent.MOUSE_DOWN, (event) => {}));

// When the mouse button is released in the area, or at the touch end
this.addEventListener(MouseEvent.MOUSE_UP, (event) => {}));
