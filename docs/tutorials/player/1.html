DisplayList

DisplayList consists of three major elements: 'Stage', 'DisplayObjectContainer', and 'DisplayObject'.

Stage

'Stage' is an object that holds all 'DisplayObject'.

DisplayObjectContainer

'DisplayObjectContainer' is a DisplayObject from which DisplayObjects can be hung.' There are four classes that inherit from 'DisplayObjectContainer' as follows.

  • Loader
  • MovieClip
  • Sprite
  • Stage

DisplayObject

'DisplayObject' is an object that performs the drawing process. There are the following nine classes that inherit from 'DisplayObject'.

  • DisplayObjectContainer
  • Loader
  • MovieClip
  • Sprite
  • Stage
  • InteractiveObject
  • Shape
  • TextField
  • Video

Adding Objects

To add a 'DisplayObject' to a 'DisplayObjectContainer', use the addChild method of the DisplayObjectContainer.

Example

const {TextField} = next2d.text;
const textField = new TextField();
textField.text = "Hello, World.";
const root = next2d.createRootMovieClip();
root.addChild(textField);
