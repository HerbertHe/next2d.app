Events when added/removed from the container.

  1. Home
  2. Tutorials
  3. Day 03

Event.ADDED

added event is an Event object that is sent when a DisplayObjectContainer is added to a DisplayObjectContainer using the addChild or addChildAt method.

Example

const {Event} = next2d.events;
this.addEventListener(Event.ADDED, onAdded);
function onAdded (event)
{
    console.log("ADDED: ", event);
}

Event.ADDED_TO_STAGE

addedToStage event is a similar event. The addedToStage event is a similar event, but it is an Event object that is sent out when a DisplayObjectContainer is added to a DisplayList and displayed on the screen.

Example

const {Event} = next2d.events;
this.addEventListener(Event.ADDED_TO_STAGE, onAddedToStage);
function onAddedToStage (event)
{
    console.log("ADDED_TO_STAGE: ", event);
}

Event.REMOVED

removed event is an Event object that is sent when a DisplayObjectContainer is removed from a DisplayObjectContainer using the removeChild and removeChildAt methods. It is also sent when a DisplayObjectContainer is repositioned from its existing position to a specified position using the addChild and addChildAt methods.

Example

const {Event} = next2d.events;
this.addEventListener(Event.REMOVED, onRemoved);
function onRemoved (event)
{
    console.log("REMOVED: ", event);
}

Event.REMOVED_FROM_STAGE

removedFromStage event is a similar event, but is an Event object that is sent out when a DisplayObjectContainer is removed from the DisplayList and disappears from the screen.

Example

const {Event} = next2d.events;
this.addEventListener(Event.REMOVED_FROM_STAGE, onRemovedFromStage);
function onRemovedFromStage (event)
{
    console.log("REMOVED_FROM_STAGE: ", event);
}
