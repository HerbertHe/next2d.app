Determine the number of DisplayObjects and child elements.

  1. Home

  3. Day 05

numChildren

numChildren property represents the number of objects in the child element. It does not include the grandchild elements.

Example

const {Shape} = next2d.display;
const shape = new Shape();
console.log(this.numChildren); // 1

contains

contains is a method to check whether the DisplayObject specified in the argument is a child element of DisplayObjectContainer. The return value is a boolean value, true if the DisplayObject is a child object or DisplayObjectContainer itself, false otherwise.

const {Sprite} = next2d.display;
const sprite1 = new Sprite();
const sprite2 = new Sprite();
const sprite3 = new Sprite();
const sprite4 = new Sprite();

sprite1.addChild(sprite2);
sprite3.addChild(sprite3);

console.log(this.contains(sprite1)); // true
console.log(this.contains(sprite2)); // true
console.log(this.contains(sprite3)); // true
console.log(this.contains(sprite4)); // false
